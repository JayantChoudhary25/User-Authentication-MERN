# User_Authentication_MERN
User Authentication (Only Backend) NodeJS, MongoDB, Express

Complete User Authentication (with Token and Hash Password)

Nodemailer is used to send reset password Token link 

## Run Command - 
npm run server

## Create config.env :-

PORT = 8000

MONGO_URI=

JWT_SECRET=

JWT_EXPIRE=10min

MAIL_SERVICE=gmail

MAIL_USERNAME=

MAIL_PASS=

MAIL_FROM=
